Simplex Algorithm Calculator

Technologies: Python

Developed an interactive user interface implementation of the Simplex Algorithm for solving linear programming problems. It allows users to input constraints and objective functions, then visualizes the step-by-step solution process.

Key Features:

  • Dynamic input system for variables and constraints
  • Real-time calculation and tableau visualization
  • Step-by-step solution breakdown for educational purposes
  • Responsive design for desktops
